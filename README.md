# Descrpción
Este proyecto consiste en plantear, cada uno, dos problemas (8 problemas en total), que se resuelven aplicando métodos de incertidumbre. Los métodos aplicados son:
- Criterio de Wald
- Criterio optimista
- Criterio de Hurwicz
- Criterio de Savage
- Criterio de Laplace
- Critero del Punto Ideal
